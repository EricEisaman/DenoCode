Deno Errors & Common Issues

In this section we'll be looking at how to fix common issues and error codes when working with Deno.

thread 'main' has overflowed its stack

This is a memory management error that shouldn't happen. It's a bug most likely in one of the module in the developers code. Best thing to do here is isolate which module is causing the issue and raise a issue on their Github repository. You can find the repositry by clicking the Readme link in the Deno Modules Directory

error: Uncaught ReferenceError: x is not defined

This issue indicates that a value is not defined. In the example above x is not a defined value. Likely a bug in the code prior to that area of execution.

Uncaught TypeError: Cannot resolve extension for ... .json

There's an issue with importing JSON files directly. Try this code instead: 

const myObj = JSON.parse(Deno.readTextFileSync('./whatever.json'));
